# DP_UserManagementMicroservice
-----------------------------------------------

1. Change Port number to : 9051
	```
	application.properties
	### Server port #########
	server.port=9051
	  
	### Context root ########
	server.contextPath=/DPUserManagementService
	```

2. Build Jar & run  
    ```
    java -jar dpuser-1.0.jar
    ```
